Super Admin
- can access all features of the system but it has no assigned branch since it is the owner thus can access all branches
- can overview the the stats of all branches (customized dashboard of all branches, provide graphs or charts)
- can add, edit, delete users & branches

Restore/Delete Function
- only super admin can delete/restore pawned items
- status:pawn - restore -> updated is_deleted=0, added to pawn table
	      - update COH (less pawned amount)
	      - permanently delete
- status: claimed - restore -> updated is_deleted=0, added to claim table
